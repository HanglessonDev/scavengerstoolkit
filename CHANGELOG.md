## [0.1.0] - 2026-02-10

### Added
- Componentes de alças de mochila (Backpack Straps)
  - Basic Backpack Straps
  - Reinforced Backpack Straps
  - Tactical Backpack Straps
- Componentes de tecido de mochila (Backpack Fabric)
  - Basic Backpack Fabric
  - Reinforced Backpack Fabric
  - Tactical Backpack Fabric
- Componente de fivela de cinto (Belt Buckle)
  - Reinforced Belt Buckle
- Arquivos de tradução em inglês e português brasileiro
- Modelos 3D para os novos itens
- Texturas para os novos itens
- Documentação completa do sistema de reciclagem

### Changed
- Atualizado nomes de componentes para seguir hierarquia consistente (Basic/Reinforced/Tactical)
- Atualizadas todas as documentações para refletir nomes atuais dos itens
- Padronizado sistema de nomenclatura de itens (STK.NomeDoItem)

### Fixed
- Corrigido erro de ortografia em "Tatical" para "Tactical" em arquivos de tradução
- Corrigido nome de modelo 3D de "BeltBuckletReinforced_Ground" para "BeltBuckleReinforced_Ground"
- Atualizados arquivos de modelo e item para manter consistência de nomenclatura

### Removed
- Texturas antigas que não estavam sendo utilizadas
